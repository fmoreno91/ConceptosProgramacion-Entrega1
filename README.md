# Proyecto: Conceptos Fundamentales de Programación - Entrega 1

Este repositorio contiene el desarrollo de la **Entrega 1 (Semana 3)** para la materia de Conceptos Fundamentales de Programación. El objetivo principal es la generación de archivos de datos pseudoaleatorios que servirán como entrada para un sistema de procesamiento de ventas.

## 🚀 Funcionalidades de esta entrega
El programa desarrollado en la clase `GenerateInfoFiles` realiza las siguientes tareas:
* **Generación de Productos:** Crea un archivo `productos.txt` con IDs, nombres y precios aleatorios.
* **Información de Vendedores:** Genera un archivo `vendedores.txt` vinculando nombres, apellidos y documentos de identidad.
* **Simulación de Ventas:** Crea archivos individuales para cada vendedor (ej. `ventas_ID.txt`) con registros de productos y cantidades vendidas.

## 📂 Estructura del Proyecto
* `src/`: Contiene el código fuente en Java (`GenerateInfoFiles.java`).
* `data/`: Carpeta donde se almacenan los archivos `.txt` generados por el programa.
* `.gitignore`: Configuración para evitar subir archivos basura al repositorio.

## 🛠️ Requisitos e Instalación
* **Lenguaje:** Java 8 o superior (Desarrollado con JDK 26).
* **IDE:** VS Code / Eclipse.
* **Ejecución:** 1. Abrir el proyecto en el IDE.
  2. Ejecutar la clase `GenerateInfoFiles.java`.
  3. Verificar los resultados en la carpeta `data/`.
